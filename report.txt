*** CAPA

  Calculadora Comercial Web 
(HTML5/CSS3/Javascript+jQuery)

*** INTRODUÇÃO

Atualmente em TI temos uma miríade de linguagens de programação e tecnologias disponíveis. Optar por uma ou outra é uma questão de projeto e por vezes de razões históricas. Contudo, toda tecnologia tem seu fundamento. Neste PPI iremos lidar com os elementos fundamentais para o desenvolvimento de aplicações Web. O uso destas tecnologias lançará a base para o aprendizado e entendimento de outras tecnologias mais complexas. Também vamos introduzir o trabalho com versionamento de arquivos, que é uma forma de trabalho em equipe, principalmente no âmbito tecnológico que gera maior produtividade. Claro, existe uma curva de aprendizado para se absorver este novo modo de trabalho e também um certo trauma a ser vencido, já que trabalhar em equipe tem elementos adicionais que devem ser considerados. Ao final do trabalho esperamos que todos os membros da equipe tenham entendido plenamento a sua parte e também as partes feitas pelos outros membros da equipe, para isso os comentários devem ser claros e efetivos.

*** OBJETIVO

Nosso objetivo será o desenvolvimento de uma Calculadora Comercial web usando HTML5, CSS3 e Javascript+jQuery. Ela rodará online ou não, e terá a mesma aparência nos diversos dispositivos hoje existentes. Para isso vamos usar as três camadas de desenvolvimento: Estrutura (HTML), Formatação (CSS) e funcionalidade (Javascript + jQuery). No final teremos esses três arquivos que juntos formarão a aplicação propriamente dita. 

*** JUSTIFICATIVA

Apesar de existirem um sem número de calculadoras já disponíveis na web, decidimos por este tema porque ele abrange praticamente todas as disciplinas ministradas neste semestre. Nosso foco contudo será no projeto e trabalho em equipe, já que é muito diferente trabalharmos sozinhos e em grupo. Para este projeto, por exemplo, introduzimos o uso do GitHub, que é o mais conhecido sistema de versionamento de arquivos. Também todos os membros da equipe receberam instruções para documentarem muito bem todo código para que este sirva de referência no futuro. Com este projeto visamos consolidar os conceitos vistos durante o curso e expandi-los para um projeto Open Source prático, ainda que simples.

*** DESENVOLVIMENTO

Por se tratar do desenvolvimento de uma aplicação, todos os comentários e código estão contidos em um repositório.  Este repositório pode ser alcançado em: https://github.com/SumareEquipeSigma/ppi-02.git

Ele é de domínio público e qualquer um pode baixar, usar, alterar e publicar novos trabalhos, desde que mencione a fonte inicial.

i*** CONCLUSÃO

Este projeto foi muito importante para destacar as diferenças entre se fazer algo sozinho e em grupo. Principalmente em se tratando de TI. A maioria de nós está acostumada a sair produzindo algo sem se preocupar com projeto ou documentação. Aprendemos que é de suma importância discutirmos em grupo a estrutura e rumos que vamos tomar, a falta de comunicação é sem dúvida o maior problema hoje na maioria dos projetos, e também a maior causa de projetos ruins. Ao final do nosso trabalho temos agora um projeto Open Source que pode usado por qualquer pessoa para aprender a trabalhar com HTML5, CSS3 e Javascript+jQuery. Esperamos no futuro poder expandir as funcionalidades desta calculadora, talvez até para uma calculadora científica, o que é totalmente possível dados os moldes em que o projeto foi realizado.
