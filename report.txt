Justificativa

Apesar de existirem um sem número de calculadoras já disponíveis na web, decidimos por este tema porque ele abrange praticamente todas as disciplinas ministradas neste semestre. Nosso foco contudo será no projeto e trabalho em equipe, já que é muito diferente trabalharmos sozinhos e em grupo. Para este projeto, por exemplo, introduzimos o uso do GitHub, que é o mais conhecido sistema de versionamento de arquivos. Também todos os membros da equipe receberam instruções para documentarem muito bem todo código para que este sirva de referência no futuro. Com este projeto visamos consolidar os conceitos vistos durante o curso e expandi-los para um projeto Open Source prático, ainda que simples.
